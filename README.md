# dietku-backend
### Installation
```
npm install
```

### Create Database
```
npx sequelize db:create
```

### Migrate database
```
npx sequelize db:migrate
```

### Seeds into database
```
npx sequelize db:seed
```

### Running
```
npm run dev
```
running at server ```http://localhost:3000```
